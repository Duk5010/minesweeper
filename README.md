# Retro Minesweeper Project
## File Purposes

### HTML
- **index.html**:  
  Contains game container, grid canvas/div, UI elements (timer, mine counter, reset button, difficulty selector)

### CSS
- **styles.css**:  
  Layout, grid system, responsive design
- **retro.css**:  
  Pixel art effects, borders, retro color palette, CRT effects (optional)

### JavaScript
- **main.js**:  
  - Initialize game on page load  
  - Set up event listeners  
  - Handle difficulty selection  
  - Connect all modules
